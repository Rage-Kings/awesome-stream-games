# Awesome Stream Games

Набор мини-игр для стримеров на Godot 4.6. Зрители участвуют в играх через чат Twitch и Kick, используя команды (например, `!join`, `!answer 2`).

## Возможности

- Подключение к чатам Twitch (IRC) и Kick (Pusher) одновременно
- Система мини-игр с автоматической ротацией и общим лидербордом
- Простое добавление новых мини-игр через базовый класс `BaseMinigame`
- Планируется интеграция с внешним API (Laravel) для хранения статистики

## Быстрый старт

```bash
# Открыть в редакторе Godot 4.6
godot --editor --path .

# Запустить проект
godot --path .
```
